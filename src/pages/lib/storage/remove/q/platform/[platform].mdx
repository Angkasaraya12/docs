Delete stored data from the storage bucket. To remove a file, you need to provide the key. `accessLevel` can be 'guest', 'private', or 'protected', defaulting to 'guest'.
Files can only be deleted for the current user. Deletion capabilities do not extend to files associated with other users.

## Guest level remove

```javascript
import { remove } from 'aws-amplify/storage';

try {
  await remove({ key: filename });
} catch (error) {
  console.log('Error ', error);
}

```

## Protected level remove

```javascript
import { remove } from 'aws-amplify/storage';

try {
  await remove({ key: filename, options: { accessLevel: 'protected' } });
} catch (error) {
  console.log('Error ', error);
}
```

## Private level remove

```javascript
import { remove } from 'aws-amplify/storage';

try {
  await remove({ key: filename, options: { accessLevel: 'private' } });
} catch (error) {
  console.log('Error ', error);
}
```
