## Prerequisites

This guide uses updated model types that feature Lazy Loading support. To follow this guide, locate `"generatemodelsforlazyloadandcustomselectionset"` in `{project-directory}/amplify/cli.json` and set the value to `true`. 

```json
{
  "features": {
    "codegen": {
      "generatemodelsforlazyloadandcustomselectionset": true
    }
  }
}
```
