If you want to delete the location history associated with a particular tracker, you can specify that as well:

<BlockSwitcher>
<Block name="Java">

```java
GeoDevice deviceId = GeoDevice.createIdTiedToUserAndDevice();
void updateLocation() {
  Amplify.Geo.deleteLocationHistory(
    deviceId,
    GeoDeleteLocationHistoryOptions.builder()
            .tracker("tracker").build(),
    () -> {
        Log.i("MyAmplifyApp", "Started tracking");
    },
    error -> {
        Log.e("MyAmplifyApp", "Start tracking failed", error);
    }
  );
}
```

</Block>
