## Install Amplify Libraries

Add the following dependencies to your `pubspec.yaml` file and install dependencies when asked:

```yaml
environment:
  sdk: ">=2.12.0 <3.0.0"

dependencies:
  flutter:
    sdk: flutter
  amplify_flutter: ^0.4.0
  amplify_datastore: ^0.4.0
```
